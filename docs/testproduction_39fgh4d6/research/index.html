Reseach Directions

Our research covers the security of entire computer stack from physical and chip security all the way up to the software security. Our projects includes but not limited to the following topics:

  • Microarchitectural Security (UARC SEC)
  • Post Quantum Cryptography (PQ CRYPT)
  • AI for Hardware Security (AI SEC)
  • CAD for Hardware Security (CAD SEC)
  • Secure Edge Devices (EDG SEC)
  • Tamper and counterfeit Detection (HWI SEC)
  • Side Channel and Fault Injection Attack and Defense (SCA SEC)
  • Photonic and Laser Assisted Physical Security (PHY SEC)
